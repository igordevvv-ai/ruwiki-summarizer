# RuWiki Summarizer

Небольшой проект на Python, который по названию статьи на Рувики (`ru.ruwiki.ru`) автоматически делает упрощённый конспект статьи для школьника 7–9 класса.

На вход:  
- название статьи, например `"Изотопы"`.

На выход:  
- сжатый, упрощённый текст,
- сохранена структура оригинала (вступление + разделы),
- текст в нейтральном, энциклопедическом стиле,
- без выдуманных фактов и чисел.
---

## 1. В чем была суть задачи?

Нужно было:

1. Взять статью с Рувики, например:  
   `https://ru.ruwiki.ru/wiki/Изотопы`
2. Автоматически:
   - получить её текст с вики-разметкой,
   - упростить текст до уровня школьника 7–9 класса,
   - сохранить основную структуру и смысл,
   - оформить результат в формате вики-статьи.
3. Подготовить решение на Python.

Важные требования:

- не придумывать новые факты и цифры;
- соблюдать стиль Википедии;
- сделать код читаемым и поддерживаемым.

---

## 2. Как подошел к решению?

Решение разбито на несколько частей:

1. **Скрипт `main.py`**  
   Точка входа. Здесь разбираются аргументы командной строки (какую статью брать, куда писать результат) и вызывается основной пайплайн.

2. **Пакет `ruwiki_summarizer` (в папке `src/`)**  
   В нём вся логика:
   - `config.py` — настройки, связанные с RuWiki (хост и User-Agent);
   - `fetch.py` — скачивание исходного текста статьи с RuWiki;
   - `parse.py` — разбор вики-разметки: выделение вступления и разделов;
   - `prompts.py` — формирование промптов для LLM;
   - `llm_client.py` — обращение к локальной LLM через Ollama;
   - `pipeline.py` — склейка всего в единый пайплайн.

Идея такая:  
**fetch → parse → prompts → LLM → сборка результата.**
